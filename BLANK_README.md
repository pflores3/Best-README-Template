## Getting Started
Installation intended for MacOS and Linux based systems
### Prerequisites
* Python 3.7 - The orchestration services are executed in a Python runtime environment.
* Pip - Cross-platform package manager for installing and managing Python packages 
* xmltodict - Python module that makes working with XML feel like you are working with JSON.
* json2xml - Simple Python Library to convert JSON to XML
  
### Installation

* Python 3.7

  ```sh
  yum install python3
  ```
* Pip
  ```sh
  yum install python3-pip
  ```
* xmltodict
  ```sh
  yum install xmltodict
  ```
 * json2xml
   ```sh
   yum install json2xml
   ```
